## Sampling Rate

- **After every task commit:** Run the per-task quick command (see Per-Task Verification Map below)
- **After every plan wave:** Run the full suite for the runtime touched (`npx vitest run` OR `cd analytics-service && pytest`)
- **After every cross-runtime wave (TS + Python both touched):** Run BOTH full suites
- **Before `/gsd-verify-work`:** Full suite must be green
- **Max feedback latency:** 45 seconds (quick); 5 minutes (full)
