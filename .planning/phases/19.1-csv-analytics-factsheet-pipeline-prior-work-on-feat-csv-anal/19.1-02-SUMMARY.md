# Phase 19.1 Plan 02: Python Analytics Summary

## One-Liner

Extend `analytics-service` to compute CSV-sourced analytics via the same
`compute_all_metrics` pipeline as exchange-backed strategies â€” fed from a
persisted `csv_daily_returns` table via a new sibling runner
`run_csv_strategy_analytics`, dispatched by the new `compute_analytics_from_csv`
worker kind with a strict 15-min watchdog over 10-min handler timeout.
